Para compilar:
Use el comando 'make' sin comillas.
Use el comando 'make help' sin comillas para ver todas las opciones.

Para ejecutar:
Una vez compilado use el comando './main' sin comillas
-El programa no necesita parámetros en consola.
-El ejecutable debe estar en la misma carpeta que las imágenes para poder procesarlas.
-El programa generará todas los modelos de ajuste a las imágenes originales y los archivos de texto que representan las tablas de ángulos y longitudes con las que se generaron los datos.
