Para compilar:
Usa el comando make (en consola) ejecuta lo siguiente:
make debug - Compila en modo debugueo, usando la bandera -g y sin usar la macro NDEBUG.
make release - Compila en modo optimización, desactivando la opción de debugueo.

Para ejecutar:
./main [I0] [In]

Ejemplo de ejecución:
./main -7 5

Para limpiar:
make clean
