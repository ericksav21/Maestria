Este es el script usado para el análisis de diversidad y fitness promedio de las ejecuciones del algoritmo genético, para usarlo simplemente hay que cambiar la variable 'path_data' a la ruta en donde se encuentren los archivos generados por el programa principal, pero de este conjunto de archivos hay que excluir el 'res_info.txt'.

*Para ejecutar el script se requiere tener instalado: numpy, matplotlib.
