Este script se usó para generar las gráficas mostradas en el reporte y adjuntadas en el presente trabajo, para ello se usó el módulo 'Matplotlib'.

Para usar el script sólamente hay que actualizar las variables 'dir_path' y 'base' que en conjunto generarán una ruta absoluta en donde se encuentran los archivos con sufijo '_stats.txt'.

El programa leerá todos los archivos y generará imágenes en formato pgn representando a las gráficas que contienen la evolución del fitness y de la temperatura a lo largo de la ejecución.

Las gráficas se generarán en la misma carpeta que el script.
